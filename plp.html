Modes

BALMUDA The Toast has five modes to heat.

Sandwich Bread Mode

Ideal for everyday thin-sliced bread and English muffins.

Artisan Bread Mode

If you want to enjoy a crisp and delicious surface, let's try this mode.

Pizza Mode

Are there leftover pizza? Use this mode to make them delicious again.

Pastry Mode

Heat croissants with this mode. It is so crisp.

Oven Mode

This mode is perfect for gratin, cookies or frozen foods.
